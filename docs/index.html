The Visitor's Agentic Sidekick

A hands-on walkthrough that shows partners how to use High Velocity Engineering, CAIRA, and the Microsoft Learn MCP to build and ship a real agentic application: a multimodal museum guide grounded live in the open Met Collection API.

Who this is for

Prove that an agent with tools beats autocomplete

Built to pitch to a technical or semi-technical partner audience. Every step is real and runnable, with the full HVE methodology (research → plan → build → review) underneath and CAIRA supplying the validated infrastructure and app scaffolding.

What you will build

A multimodal guide, grounded in open data

Themed tours

One sentence in, a curated, sequenced, narrated tour out, across departments and cultures.

Vision explanations

GPT-4o vision reads real artwork and explains it in plain, friendly language.

Live retrieval

Agentic tool-calling against the Met API, no baked-in dataset, no API key.

The three building blocks

What powers the walkthrough

HVE Core

CAIRA
