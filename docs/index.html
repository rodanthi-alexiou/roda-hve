HVE Practices Lab

Ship production AI apps faster with proven engineering practices

High Velocity Engineering (HVE) brings structured methodology to GitHub Copilot workflows. Instead of raw prompting, you follow Research → Plan → Implement → Review — producing verified, traceable code that reaches production faster and with fewer defects.

This lab demonstrates HVE in action: a real agentic app built end-to-end with HVE Core, CAIRA accelerators, and the Microsoft Learn MCP — all driven by GitHub Copilot.

Why HVE matters

Structured AI-assisted engineering across the SDLC

HVE is not just about writing code faster. It integrates AI pair programming into every phase of the software development lifecycle — from requirements research to deployment review — with guardrails that ensure quality, security, and traceability.

Research

Ground decisions in real docs and APIs before writing a line of code. The Learn MCP verifies everything against current Microsoft documentation.

Plan

Generate checkboxed implementation plans with clear task decomposition, dependencies, and acceptance criteria.

Implement

Write production code using CAIRA reference architectures and HVE agents, not blank-slate prompting.

Review

Automated validation reports confirm the implementation matches the plan. Every artefact is traceable.

  • A GitHub account with GitHub Copilot enabled
  • An Azure subscription where you can create Azure AI Foundry and Container Apps resources
  • Permission to create role assignments (CAIRA IaC assigns least-privilege roles)
No key for the Met

The Met Collection API needs no account or key. All requests are anonymous and public.

Install HVE Core

For VS Code, install the extension:

ise-hve-essentials.hve-core-all

For the CLI:

copilot plugin marketplace add microsoft/hve-core
copilot plugin install

Reload VS Code, then confirm HVE agents (rpi-agent, task-researcher) appear in the Copilot Chat picker.

Install CAIRA skill + Learn MCP

CAIRA skill:

npx skills add github.com/microsoft/CAIRA/skills

Learn MCP — add to .vscode/mcp.json:

{
  "servers": {
    "microsoft-learn": {
      "type": "http",
      "url": "https://learn.microsoft.com/api/mcp"
    }
  }
}

This gives Copilot live access to microsoft_docs_search, microsoft_code_sample_search, and microsoft_docs_fetch.
