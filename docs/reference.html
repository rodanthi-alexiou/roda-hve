Met Collection API

API reference

The Met Collection API is open, keyless, and returns CC0 public-domain metadata.

PropertyValue
Base URLhttps://collectionapi.metmuseum.org/public/collection/v1
AuthenticationNone
LicenseCC0 (public domain) for flagged objects
Rate limit80 requests per second
Objects470,000+

Endpoints

EndpointReturnsNotes
/searchAn array of object IDs onlyMust fan out to /objects/{id} for details
/objects/{id}Full metadata and image URLsBatch and cache these calls
/objectsAll object IDs and a totalRarely needed for the demo
/departmentsThe list of departmentsPowers themed, cross-department tours

The N+1 guardrail

Because /search returns only IDs, retrieving details is an N+1 problem. The tool layer solves it with a batched, cached fetch that caps concurrency and filters to public-domain works with images. See Step 3 for the implementation.
